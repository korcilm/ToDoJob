## General info
This project is a business management system project in N-tier architecture. There are separate areas for manager and employee in the project (role-based authorization).
	
## Technologies
Project is created with:
* .Net Core: 3.1
* Entity Framework
* Fluent Validation
* MSSQL
* AutoMapper
